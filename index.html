Game Overview

GhostHunter is a retro, halloween themed action maze chase game. The game follows the main character, referred to as “child”, who wakes suddenly in a maze-like graveyard that is guarded by the Grim Reaper and his vengeful ghosts. In order to escape, the main character must save all the ghosts stuck in the graveyard before Grim Reaper catches him.
